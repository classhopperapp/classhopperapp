# Üniversite Hayatında Sınıf Atla!

Klanlara katıl, gerçek projelerde yerini al, networkünü genişlet ve şirketleri cezbet.

# Hakkımızda

Classhopper, üniversite öğrencilerini gerçek projeler, ekip çalışmaları ve şirketlerle buluşturan bir platformudur. Amacımız, öğrencilerin kampüs hayatında sadece derslerle değil, gerçek dünya deneyimleriyle de kendilerini geliştirmelerini sağlamak. Klan sistemi sayesinde farklı disiplinlerden öğrenciler bir araya gelir, projeler üretir ve geleceğe hazır hale gelir.
